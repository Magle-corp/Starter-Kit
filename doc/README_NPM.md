## Requis

Assurez-vous que les ports suivants sont disponibles :
- 3000
- 1337
- 3306
- 8081

## Installation

```bash
npx @magle-corp/starter <nom_du_projet>
cd <nom_du_projet>
cp .env.example
docker-compose up
```
